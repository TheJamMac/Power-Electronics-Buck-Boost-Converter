# Power Electronics Project: DC-DC Buck and Boost Converter (LTspice)

This project is about DC-DC converter design in power electronics.
The aim of the project is to study how Buck and Boost converters work and how we can simulate them in LTspice.

In this project I design simple Buck and Boost converter circuits. I use LTspice to see the voltage, current and switching behavior. I also look at efficiency and output voltage ripple.

The project also includes basic PCB preparation for the converter circuit.

## Project Goals

* Understand how DC-DC converters work
* Design Buck converter circuit
* Design Boost converter circuit
* Simulate the circuits in LTspice
* Observe voltage and current waveforms
* Analyze efficiency and output ripple

## Tools Used

* LTspice
* Power Electronics concepts
* Basic PCB design tools
* GitHub for project documentation
