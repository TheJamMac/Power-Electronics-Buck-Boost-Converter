# DC-DC Converter Theory

DC-DC converters are widely used in power electronics to convert one DC voltage level to another.

Two common types of DC-DC converters are:

Buck Converter  
Boost Converter

## Buck Converter

A Buck converter is used to step down the input voltage.

Example:
12V input → 5V output

Main components:
- MOSFET (switch)
- diode
- inductor
- capacitor

The switching operation controls the output voltage.

## Boost Converter

A Boost converter is used to increase the input voltage.

Example:
5V input → 12V output

The inductor stores energy when the switch is ON and releases it when the switch is OFF.

This energy transfer increases the output voltage.
